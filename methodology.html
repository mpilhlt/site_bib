The SCC Explorer was developed for several reasons. First of all we aimed to provide not only classical bibliographical collection, but also make it more suitable for custom search. For this task, we developed a particular onthology for the large bibliographical dataset and organized in accordance with a hierarchical order that also reflect the hierarchy of the Congregation of the Council documents.

Challenges
Problems and their solutions

This project aims to provide an interactive searching opportunities among a large bibliographical dataset that could be used for any other library. Realization of this task faced several challenges as on the theoretical as on the technical level.

read more close

First of all, we aimed to organize the large massive of titles, which one one hand, should demonstrate the logical structure of the Congregation of the Council related subjects, and on the other hand, this system should not be overwhelming for readers and for the machines. For this reason, among languages and frameworks we choose R, because of the libraries for both aesthetically pleasant visualisation of graphic representation and simplicity to compile and run everything inside the shiny framework. Secondly, the core of this application is a capacity to render the search results in the .bib format, which allows readers to add these titles into the library software. This manipulation became possible because of a double conversion of the original .bib file, which consists of the data, provided by the group members, to the data table and vice versa for rendering a file with selected inputs for downloading.

Thirdly, we also aimed to provide the option for the quantitative research and graphic visualisation of the bibliographical tendencies. For this reason we used ggplot2 and data tables for makings interactive graphs. A user can set up several filters to particulate the data sample and choose a graphic representation among lines, bars, pies and dots.

Resources Selection
Resources we use

These resources were obtained within the personal research of the group members that include such subjects as canon law, constitutional law, Church history, historical geography, religious studies and many others.

read more close

We decided to put the database into the framework of the Congregation of the Council and closely related subjects, which are strongly connected with the Congregation. Therefore, we do not include the general titles about the Church History or Canon Law, if they were not directly addressed to the Congregation's competences.
